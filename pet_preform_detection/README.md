Основной файл для запуска main. 
Настройки для работы с модулем config.py

Перед запуском необходимо:
1. указать путь к видеофайлу PATH_VIDEO
2. способ детекции YOLO или CV2(OpenCV)
3. указать настройки по необходимости: форматы выгружаемых файлов, минимальный порог уверенности для предсказаний модели и т. д.

Запуск в vcs: 
1. перейти в директорию cd.\pet_preform_detection
2. выбрать интерпретатор: ctrl+shift+p -> Python:Select Interpretator ->Python 3.10.0
3. запустить в терминале python main.py

Модель для YOLO обучена и находится в каталоге model/best.pt
